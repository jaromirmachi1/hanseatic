## Hanseatic (React + Vite)

Minimalist, high-end one-page site for **Hanseatic** with calm GSAP + ScrollTrigger motion and optional Lenis smooth scrolling.

### Run locally

```bash
npm install
npm run dev
```

### Production build

```bash
npm run build
npm run preview
```

### Notes
- The original static prototype is preserved in `legacy-static/`.
- Images are local SVG placeholders in `src/assets/`.
